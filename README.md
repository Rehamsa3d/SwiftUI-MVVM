# SwiftUI-MVVM
A simple SwiftUI + MVVM app that fetches users from a remote API and displays them in a list.

## Tech Stack
- SwiftUI
- MVVM
- async/await
- URLSession

## Features
- Fetch users from DummyJSON API
- Display user image, name and age
- Loading & error handling
- Clean MVVM architecture

## API
https://dummyjson.com/users

## How to Run
1. Clone the repo
2. Open in Xcode 15+
3. Run on Simulator or device

## Architecture
- Features
  - Users (View / ViewModel / Model)
- Services
  - UserService
